# Weather API with Current Time
Designed a page that declared the current time of the user and allowed them to add the Weather of their city through the use of a RESTFUL Weather API.


### Learning Outcomes
- [x] ES6 Classes
- [x] Fetch
- [x] HTTP Methods
- [x] Promises
- [x] Async/Await
- [x] Date Object
- [x] Responsiveness
- [x] Local Storage
